# Invitacion-15-anos
# 🎀 Invitación 15 Años – María Alejandra

Invitación digital temática **Cuento de Hadas** para los 15 años de María Alejandra.

## ✨ Características
- Diseño elegante en tonos dorados
- Partículas animadas de polvo de hadas
- Cuenta regresiva en tiempo real
- Botón de confirmación por WhatsApp
- Totalmente responsive (funciona en celular y PC)
- Publicable gratis con GitHub Pages

## 📁 Archivos
| Archivo | Descripción |
|---------|-------------|
| `index.html` | Estructura de la invitación |
| `style.css` | Estilos y diseño dorado |
| `script.js` | Animaciones y cuenta regresiva |

## ⚙️ Personalización
Antes de publicar, edita en `index.html`:
- **Fecha del evento** (sección de detalles)
- **Hora y lugar**
- **Número de WhatsApp** para confirmación

En `script.js`, cambia la fecha en:
```js
const EVENT_DATE = new Date('2025-07-12T19:00:00');
```

## 🌐 Publicar con GitHub Pages
1. Sube los archivos a tu repositorio
2. Ve a **Settings → Pages**
3. Selecciona la rama `main`
4. ¡Tu invitación estará en línea!
